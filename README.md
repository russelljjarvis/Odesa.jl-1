# Odesa.jl-1
Experimental build of Odesa in julia

To Install

```
] add "https://github.com/russelljjarvis/Odesa.jl-1"
```

Or

```
using Pkg
Pkg.add(url="https://github.com/russelljjarvis/Odesa.jl-1")
```
